# @unified/dashboard

Dashboard app for Unified Shipping Platform (Next.js + Tailwind + Vitest).

## Quickstart

Install dependencies from repo root:

```sh
pnpm install
```

Run the app:

```sh
pnpm --filter @unified/dashboard dev
```

Build and start:

```sh
pnpm --filter @unified/dashboard build
pnpm --filter @unified/dashboard start
```

Run tests:

```sh
pnpm --filter @unified/dashboard test
```

Healthcheck is available at `/api/health`.
